# Topic Ideas Index

This folder collects strong topics for testing the CrewAI + LaTeX generator. The ideas are intentionally varied so the project can demonstrate article/book generation across culture, software engineering, fashion, food, sport, healthcare, Hebrew layout, English layout, and BiDi text.

| File | Style | Language | Focus |
|---|---|---|---|
| `food_hebrew_article.md` | Article | Hebrew | Family recipes, culinary memory, consent, and cultural preservation |
| `computer_science_english_article.md` | Article | English | Self-healing repositories and agentic technical-debt repair |
| `fashion_hebrew_book.md` | Book | Hebrew | Algorithmic wardrobes, identity, sustainability, and style agents |
| `sports_english_book.md` | Book | English | AI agents, fan emotion, analytics, and global sport |
